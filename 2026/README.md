# Portfolio 2026

A modern, animated personal portfolio website featuring the Everforest color scheme.

## Features

- Smooth animations and micro-interactions
- Custom cursor with trailing effect
- Scroll progress indicator
- Responsive design
- Card hover effects
- Floating background shapes with parallax
- Intersection Observer for scroll-triggered animations

## Colors

Based on the Everforest Dark (Medium) color palette:
- Background: #232A2E - #3D484D
- Foreground: #D3C6AA
- Accents: Green (#A7C080), Blue (#7FBBB3), Purple (#D699B6), Orange (#E69875)

## Usage

Open `index.html` in any modern web browser.
